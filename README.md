> **Compare 400+ AI models from all providers. Find the cheapest option instantly.**

TokenTax is a free, open-source tool to compare pricing across **400+ AI models** from OpenAI, Anthropic, Google Gemini, xAI Grok, Mistral, DeepSeek, and more.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📊 **400+ Models** | All major AI providers |
| 💰 **Price Comparison** | Sorted cheapest to expensive |
| 🔍 **Search & Filter** | By name or provider |
| 📥 **CSV Export** | Download results |
| 📱 **PWA** | Installable, works offline |
| 📈 **Stats** | Track calculations & savings |
