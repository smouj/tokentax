> **Compare 400+ AI models from ALL providers in real-time. Find the cheapest option for your prompts instantly.**

TokenTax is a free, open-source web tool that lets you compare pricing across **400+ AI models** from every major provider — **OpenAI, Anthropic, Google Gemini, xAI Grok, Mistral, DeepSeek, Groq, Together.ai, Fireworks, DeepInfra, AWS Bedrock, Azure, Llama, Qwen**, and many more.

**Data is fetched live from LiteLLM** — the most complete and up-to-date AI pricing database in the world. Prices are updated weekly by the community.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📊 **400+ Models** | Compare models from all major AI providers |
| 🔍 **Real-time Search** | Filter by name: "grok", "claude", "llama", "gemini"... |
| 🏷️ **Provider Filter** | Select one or multiple providers |
| 💰 **Sorted by Price** | Automatically sorted from cheapest to most expensive |
| ⚡ **Instant Calculation** | Calculate exact cost of your specific prompt |
| 🌐 **Multi-language** | Available in 7 languages |
| 📱 **Responsive Design** | Works perfectly on desktop and mobile |
| 🔄 **Live Data** | Prices update automatically from LiteLLM |
| 🌓 **Dark/Light Mode** | Toggle theme with persistence |
| 📥 **CSV Export** | Download results as CSV |
| 🔢 **Token Counter** | Real-time char/token estimation |
| 🔃 **Column Sorting** | Sort by any column |
| ⌨️ **Keyboard Shortcuts** | Ctrl+Enter to calculate |
| 📋 **Copy Results** | Copy top result to clipboard |
| 🍞 **Toast Notifications** | Visual feedback |

## 🛠️ Usage

1. Open **[https://smouj.github.io/tokentax/](https://smouj.github.io/tokentax/)**
2. Enter your prompt in the text area
3. Set expected output tokens
4. Click **"Calculate costs with +400 models"** (or press `Ctrl+Enter`)
5. See all models sorted by price — find the cheapest option! 🔥

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + Enter` | Calculate costs |
