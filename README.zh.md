> **比较来自所有提供商的400+ AI模型。即刻找到最便宜的选项！**

TokenTax是一个免费的、开源的网页工具，让你能够实时比较**400+ AI模型**的价格。

---

## ✨ 功能

| 功能 | 描述 |
|------|------|
| 📊 **400+ 模型** | 比较OpenAI、Anthropic、Google、xAI、Mistral、DeepSeek等 |
| 💰 **价格比较** | 按价格从低到高排序 |
| 🔍 **搜索和筛选** | 按名称搜索，按提供商筛选 |
| ⚡ **即时计算** | 计算你的具体提示词的成本 |
| 📥 **CSV导出** | 下载比较结果 |
| 🌓 **深色/浅色模式** | 切换主题 |
| 📱 **PWA** | 可安装，离线可用 |
| 📈 **统计数据** | 跟踪计算和节省 |

## 🛠️ 使用方法

1. 打开[演示](https://smouj.github.io/tokentax/)
2. 输入你的提示词
3. 设置输出tokens
4. 点击**计算**或按`Ctrl+Enter`
5. 找到最便宜的模型！🔥
