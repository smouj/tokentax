> **实时比较来自所有提供商的400+ AI模型。即刻找到最便宜的提示词选项！**

TokenTax是一个免费的、开源的网页工具，让你能够比较**400+ AI模型**的价格，涵盖所有主要提供商 — **OpenAI、Anthropic、Google Gemini、xAI Grok、Mistral、DeepSeek、Groq、Together.ai、Fireworks、DeepInfra、AWS Bedrock、Azure、Llama、Qwen** 等等。

**数据直接从LiteLLM实时获取** — 世界上最完整、最新的AI定价数据库。价格每周由社区更新。

---

## ✨ 功能

| 功能 | 描述 |
|------|-------------|
| 📊 **400+ 模型** | 比较所有AI提供商的模型 |
| 🔍 **实时搜索** | 按名称筛选："grok"、"claude"、"llama"、"gemini"... |
| 🏷️ **提供商筛选** | 选择一个或多个提供商 |
| 💰 **按价格排序** | 自动从最便宜到最贵排序 |
| ⚡ **即时计算** | 计算你的具体提示词的准确成本 |
| 🌐 **多语言** | 支持7种语言 |
| 📱 **响应式设计** | 在移动设备和桌面上完美运行 |
| 🔄 **实时数据** | 价格从LiteLLM自动更新 |

## 🛠️ 使用方法

1. 打开 **[https://smouj.github.io/tokentax/](https://smouj.github.io/tokentax/)**
2. 在文本区域输入你的提示词
3. 设置预期的输出token数
4. 点击 **"用+400个模型计算成本"**
5. 查看所有按价格排序的模型 — 找到最便宜的选项！🔥
