> **Compare mais de 400 modelos de IA de todos os provedores. Encontre a opção mais barata para seus prompts instantaneamente!**

TokenTax é uma ferramenta web gratuita e de código aberto que permite comparar preços de **mais de 400 modelos de IA** de todos os principais provedores em tempo real.

---

## ✨ Características

| Característica | Descrição |
|----------------|-------------|
| 📊 **400+ Modelos** | Compare OpenAI, Anthropic, Google, xAI, Mistral, DeepSeek... |
| 💰 **Comparação de preços** | Ordenado do mais barato |
| 🔍 **Pesquisa e filtro** | Pesquise por nome ou filtre por provedor |
| ⚡ **Cálculo instantâneo** | Calcule o custo exato para seu prompt |
| 📥 **Exportar CSV** | Baixe os resultados |
| 🌓 **Modo escuro/claro** | Alterne entre temas |
| 📱 **PWA** | Instalável, funciona offline |
| 📈 **Estatísticas** | Acompanhe cálculos e economias |

## 🛠️ Como usar

1. Abra a [demo](https://smouj.github.io/tokentax/)
2. Digite seu prompt
3. Defina os tokens de saída
4. Clique em **Calcular** ou pressione `Ctrl+Enter`
5. Encontre o modelo mais barato! 🔥
