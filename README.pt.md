> **Compare mais de 400 modelos de IA de TODOS os provedores em tempo real. Encontre a opção mais barata para seus prompts instantaneamente!**

TokenTax é uma ferramenta web gratuita e de código aberto que permite comparar preços de **mais de 400 modelos de IA** de todos os principais provedores — **OpenAI, Anthropic, Google Gemini, xAI Grok, Mistral, DeepSeek, Groq, Together.ai, Fireworks, DeepInfra, AWS Bedrock, Azure, Llama, Qwen** e muitos mais.

**Os dados são buscados ao vivo do LiteLLM** — o banco de dados de preços de IA mais completo e atualizado do mundo. Os preços são atualizados semanalmente pela comunidade.

---

## ✨ Características

| Característica | Descrição |
|----------------|-------------|
| 📊 **400+ Modelos** | Compara modelos de todos os provedores de IA |
| 🔍 **Busca em tempo real** | Filtra por nome: "grok", "claude", "llama", "gemini"... |
| 🏷️ **Filtro por provedor** | Selecione um ou vários provedores |
| 💰 **Ordenado por preço** | Ordenado automaticamente do mais barato |
| ⚡ **Cálculo instantâneo** | Calcula o custo exato do seu prompt específico |
| 🌐 **Multilíngue** | Disponível em 7 idiomas |
| 📱 **Design responsivo** | Perfeito no mobile e desktop |
| 🔄 **Dados ao vivo** | Os preços se atualizam automaticamente do LiteLLM |

## 🛠️ Como usar

1. Abra **[https://smouj.github.io/tokentax/](https://smouj.github.io/tokentax/)**
2. Digite seu prompt na área de texto
3. Defina os tokens de saída esperados
4. Clique em **"Calcular custos com +400 modelos"**
5. Veja todos os modelos ordenados por preço — encontre a opção mais barata! 🔥
