> **Confronta oltre 400 modelli AI da tutti i fornitori. Trova l'opzione più economica per i tuoi prompt istantaneamente!**

TokenTax è uno strumento web gratuito e open-source che ti permette di confrontare i prezzi di **oltre 400 modelli AI** da tutti i principali fornitori in tempo reale.

---

## ✨ Caratteristiche

| Caratteristica | Descrizione |
|----------------|-------------|
| 📊 **400+ Modelli** | Confronta OpenAI, Anthropic, Google, xAI, Mistral, DeepSeek... |
| 💰 **Confronto prezzi** | Ordinato dal più economico |
| 🔍 **Ricerca e filtro** | Cerca per nome o filtra per fornitore |
| ⚡ **Calcolo istantaneo** | Calcola il costo esatto per il tuo prompt |
| 📥 **Esporta CSV** | Scarica i risultati |
| 🌓 **Modalità scuro/chiaro** | Cambia tema |
| 📱 **PWA** | Installabile, funziona offline |
| 📈 **Statistiche** | Traccia calcoli e risparmi |

## 🛠️ Utilizzo

1. Apri la [demo](https://smouj.github.io/tokentax/)
2. Inserisci il tuo prompt
3. Imposta i token di output
4. Clicca su **Calcola** o premi `Ctrl+Enter`
5. Trova il modello più economico! 🔥
