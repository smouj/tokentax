## ✨ Funzionalità

- 📊 **400+ Modelli** - OpenAI, Anthropic, Google, xAI, Mistral, DeepSeek...
- 💰 **Confronto prezzi** - Ordinato dal più economico al più caro
- 🔍 **Cerca e filtra** - Cerca per nome o filtra per provider
- ⚡ **Auto-calcolo** - Calcola mentre scrivi
- 📥 **Export CSV/PDF** - Scarica i risultati
- ⭐ **Preferiti** - Salva modelli preferiti
- 💱 **Multi-valuta** - Supporto USD / EUR / GBP
- ⌨️ **Scorciatoie** - `Ctrl+Invio` calcola, `Esc` cancella
- 🌓 **Modalità scuro/chiaro** - Alterna tra i temi
- ⬛ **Modalità OLED** - Nero puro per display OLED
- 👁️ **Alto contrasto** - Modalità accessibilità
- 📱 **PWA** - Installabile, funziona offline
- 🐷 **Calcolatore risparmio** - Mostra risparmio potenziale
- 🔗 **Condividi link** - Condividi URL con impostazioni attuali
- ⛶ **Schermo intero** - Modalità schermo intero (F11)
- ❓ **Aiuto** - Premi ? per vedere le scorciatoie
