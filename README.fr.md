> **Comparez plus de 400 modèles d'IA de tous les fournisseurs. Trouvez l'option la moins chère pour vos prompts instantanément.**

TokenTax est un outil web gratuit et open-source qui vous permet de comparer les prix de **plus de 400 modèles d'IA** de tous les principaux fournisseurs en temps réel.

---

## ✨ Fonctionnalités

| Fonctionnalité | Description |
|----------------|-------------|
| 📊 **400+ Modèles** | Comparez OpenAI, Anthropic, Google, xAI, Mistral, DeepSeek... |
| 💰 **Comparaison de prix** | Trié du moins cher au plus cher |
| 🔍 **Recherche et filtre** | Recherchez par nom ou filtrez par fournisseur |
| ⚡ **Calcul instantané** | Calculez le coût exact pour votre prompt |
| 📥 **Export CSV** | Téléchargez les résultats |
| 🌓 **Mode sombre/clair** | Basculez entre les thèmes |
| 📱 **PWA** | Installable, fonctionne hors ligne |
| 📈 **Statistiques** | Suivi des calculs et économies |

## 🛠️ Utilisation

1. Ouvrez la [démo](https://smouj.github.io/tokentax/)
2. Entrez votre prompt
3. Définissez les tokens de sortie
4. Cliquez sur **Calculer** ou appuyez sur `Ctrl+Enter`
5. Trouvez le modèle le moins cher ! 🔥
