> **Comparez plus de 400 modèles d'IA de TOUS les fournisseurs en temps réel. Trouvez l'option la moins chère pour vos prompts instantanément !**

TokenTax est un outil web gratuit et open-source qui vous permet de comparer les prix de **plus de 400 modèles d'IA** de tous les principaux fournisseurs — **OpenAI, Anthropic, Google Gemini, xAI Grok, Mistral, DeepSeek, Groq, Together.ai, Fireworks, DeepInfra, AWS Bedrock, Azure, Llama, Qwen** et bien d'autres.

**Les données sont récupérées en direct de LiteLLM** — la base de données de prix IA la plus complète et actualizada du monde. Les prix sont mis à jour hebdomadairement par la communauté.

---

## ✨ Fonctionnalités

| Fonctionnalité | Description |
|-----------------|-------------|
| 📊 **400+ Modèles** | Comparez des modèles de tous les fournisseurs d'IA |
| 🔍 **Recherche en temps réel** | Filtrez par nom : "grok", "claude", "llama", "gemini"... |
| 🏷️ **Filtre par fournisseur** | Sélectionnez un ou plusieurs fournisseurs |
| 💰 **Trié par prix** | Trié automatiquement du moins cher au plus cher |
| ⚡ **Calcul instantané** | Calculez le coût exact de votre prompt spécifique |
| 🌐 **Multilingue** | Disponible en 7 langues |
| 📱 **Design responsive** | Parfait sur mobile et desktop |
| 🔄 **Données en direct** | Les prix se mettent à jour automatiquement depuis LiteLLM |

## 🛠️ Utilisation

1. Ouvrez **[https://smouj.github.io/tokentax/](https://smouj.github.io/tokentax/)**
2. Entrez votre prompt dans la zone de texte
3. Définissez les tokens de sortie attendus
4. Cliquez sur **"Calculer les coûts avec +400 modèles"**
5. Voyez tous les modèles triés par prix — trouvez l'option la moins chère ! 🔥
