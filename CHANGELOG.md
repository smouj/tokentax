# TokenTax AI PRO - Changelog

All notable changes to this project will be documented in this file.

## [v1.4.0] - 2026-03-08

### Added
- 🔃 **Column sorting** - Click headers to sort by provider, model, input, output, or cost
- 🍞 **Toast notifications** - Visual feedback for copy, save, errors
- ⌨️ **Keyboard shortcuts** - Ctrl+Enter to calculate
- 📝 **Quick examples** - Buttons to load sample prompts (code, summary)
- 📋 **Copy top result** - Copy best result to clipboard
- ✕ **Clear prompt** - Quick button to clear input
- 🎯 **Professional loader** - Animated loading spinner
- 📊 **Enhanced footer** - Branding
